## プロジェクト概要

**AriaDocs** は単なるドキュメントテンプレートではありません。これはモダンなウェブ開発のための **完全なツールキット** です。主な特徴は以下の通りです：

- **Markdown & MDX 対応**：Markdown で簡単にドキュメントを作成し、MDX を使用してインタラクティブなコンテンツを追加できます。
- **カスタマイズ可能なテーマ**：ミニマリストで ShadCN にインスパイアされたテーマを採用し、簡単にスタイルを変更可能。
- **SEO 最適化**：各ページは SEO に対応し、検索エンジンに適切にインデックスされます。
- **インタラクティブなコードブロック**：美しくデザインされた言語別のコードブロックを提供。

## 技術とライブラリ

AriaDocs は最新技術を活用し、高いパフォーマンスと柔軟性を提供します：

- **Next.js 14** - 本番環境向けに最適化された強力な React フレームワーク。
- **Tailwind CSS** - ユーティリティファーストのスタイリングで、素早く美しいデザインを実現。
- **Shadcn-UI** - エレガントでアクセシブルなコンポーネントを提供。
- **next-mdx-remote** - 動的でインタラクティブな Markdown コンテンツをサポート。
